# Project-Sync 🚀

A real-time collaborative workspace and agile project management platform inspired by modern tools like Jira and Trello.


## ✨ Features

- Real-time team collaboration
- Drag & drop task management
- Agile project workspace
- Team chat functionality
- JWT Authentication
- Google OAuth 2.0 Login
- Stripe payment integration
- Responsive dashboard UI
- Context API state management
- MongoDB data persistence


## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Context API
- CSS / Tailwind (if used)

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.io
- JWT Authentication

### Integrations
- Google OAuth
- Stripe API
